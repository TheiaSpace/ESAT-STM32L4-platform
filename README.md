# Arduino Core for Theia Space's ESAT Boards (STM32L4)

These are the support files needed to use Theia Space's STM32L4-based
ESAT boards with Arduino.

Use Arduino 1.8.0 or newer.


## Supported boards

ESAT ADCS.
